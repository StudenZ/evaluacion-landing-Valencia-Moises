# Evaluación Práctica Unidad 2 - Programación Web

Este repositorio contiene el proyecto de la landing page desarrollado para la evaluación práctica de la Unidad 2.

*   **Estudiante:** Moisés Elías Valencia Ramírez
*   **Asignatura:** Programación Web
*   **Tema asignado:** Repostería y Postres Artesanales
*   **Marca del proyecto:** Dulce Tentación

## Descripción del Proyecto
Es una landing page responsive de una pastelería y repostería artesanal local llamada "Dulce Tentación". El objetivo es mostrar los postres disponibles (cupcakes, macarons, pasteles, brownies, trufas, galletas) y permitir que los clientes realicen consultas mediante un formulario de contacto.

## Tecnologías utilizadas
*   HTML5 Semántico
*   CSS3 (Variables en :root, Flexbox y CSS Grid para adaptabilidad)
*   Bootstrap 5 (para la estructura del formulario y navegación)
*   Iconos de Bootstrap Icons
